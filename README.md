# cuckoo

定时提醒工具

## 依赖的软件

在cuckoo的运行过程中需要用到下列的第三方软件，请先安装

- [ControlPlane](https://www.controlplaneapp.com/)
- [alerter](https://github.com/vjeantet/alerter)
